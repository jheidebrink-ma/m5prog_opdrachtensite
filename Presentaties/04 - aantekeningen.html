Database verbinding

Om ervoor te zorgen dat je de database kunt beheren heb je 2 opties:
1- installatie van phpmyadmin.
PhpMyAdmin is een web based tool en maakt op dezelfde manier verbinding als dat je met je web applicatie doet
- Download en plaats deze je in je public folder
- Kopieer config.sample.inc.php naar config.inc.php
- pas in dit bestand de host aan naar de naam van je database container, bv: 
$cfg['Servers'][$i]['host'] = 'mariadb';
- ga in he browser naar de betreffende folder, bijvoorbeeld
http://localhost:8080/phpMyAdmin/
Hier kun je inloggen met de gegevens die in je docker-compose.yml staan onder het stukje van je database:
MYSQL_USER: db_user
MYSQL_PASSWORD: mijn_p@ss#
2- HeidiSQL of Sequel Ace
Voor deze tool moet je eerst de poort naar je ddatabase open zetten, dit doe je in je docker-compose.yml file.
Plaats in het database gedeelte bijvoorbeeld de volgende code: 
ports:
- 3307:3306
Build nu docker opnieuw: Run dit commando in de terminal: 
docker-compose up --build
Nadat je docker in de terminal opnieuw gebuild hebt kun je verbinding maken met de volgende gegevens: host: localhost
port: 3307
user: je MYSQL_USER
pass: je MYSQL_PASSWORD


Binnen een van deze tools kun je nu een tabel aanmaken die je voor je project gaat gebruiken.
Let op dat je niet de database: information_schema verwijdert, deze bevat namelijk alle gegevens die mysql nodig heeft voor de database.
